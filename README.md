# Ecommerce Website

A full-featured ecommerce website built with PHP, MySQL, Bootstrap, and FontAwesome. This project allows users to browse products, add items to their cart, place orders, and manage their accounts. Admins can manage products, categories, brands, and view orders.

## Features
- User registration and login
- Product listing, search, and filtering by category/brand
- Shopping cart with add, update, and remove functionality
- Order placement and payment confirmation
- User profile management (edit, delete account)
- Admin dashboard for managing products, categories, brands, and orders
- Responsive design using Bootstrap

## Technologies Used
- PHP & MySQL
- Bootstrap 5
- FontAwesome
- JavaScript (for UI interactions)

## Setup Instructions
1. Clone or download the repository to your local machine.
2. Import the provided SQL database into your MySQL server.
3. Configure your database connection in `includes/connect.php`.
4. Place all project files in your web server directory (e.g., `htdocs` for XAMPP).
5. Start your web server and access the site via `http://localhost/Ecommerce website/`.

## Usage
- Register as a user to start shopping.
- Browse products, add to cart, and checkout.
- Admins can log in to manage products, categories, brands, and view orders.

## Folder Structure
- `/admin_area` - Admin dashboard and management pages
- `/user_area` - User profile, orders, and account management
- `/functions` - Common PHP functions
- `/includes` - Database connection and footer
- `/images` - Product and site images
- `/style.css` - Custom styles

## Notes
- Ensure your PHP and MySQL server are running.
- Update database credentials in `connect.php` as needed.
- For payment integration, add your payment gateway logic in `confirm_payment.php`.
